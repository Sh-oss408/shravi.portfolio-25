# Shravi Shukla - Personal Portfolio Website

## Tools Used
- HTML5 (semantic tags for structure)
- CSS3 (custom styles in styles.css)
- Tailwind CSS (for layout, responsiveness, and utilities)
- Font Awesome (for icons)
- Media (profile image, audio, video)

## Features Implemented
1. **Multi-page Portfolio**
   - index.html (Home + About + Demo video)
   - projects.html (Projects overview with table + featured projects)
   - contact.html (Contact form + contact details)

2. **Semantic HTML**
   - Used <header>, <nav>, <section>, <article>, <aside>, <footer> consistently.

3. **Navigation**
   - Consistent nav bar across all pages with internal links.
   - External resume link included.

4. **Contact Form**
   - Name, email, subject, and message fields.
   - Required and validation attributes.
   - Submit and reset buttons.

5. **Tables and Lists**
   - Projects table in projects.html.
   - Ordered list (skills) and unordered list (tools) in index.html.

6. **Media Embedding**
   - Profile image.
   - Background audio.
   - Project demo video.

7. **Styling**
   - Tailwind CSS utilities for layout and responsiveness.
   - styles.css for theme, buttons, hover effects, and custom look.

8. **Responsiveness**
   - Fully responsive using Tailwind breakpoints (sm, md, lg).
   - Looks good on mobile, tablet, and desktop.

9. **Animations & Hover Effects**
   - Buttons animate on hover.
   - Profile image hover zoom.
   - Project cards with hover shadow effect.
